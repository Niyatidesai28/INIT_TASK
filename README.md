## Overview

A beginner-friendly Python project focused on data analysis and manipulation. It covers the full process — from loading and cleaning CSV data to performing simple business analysis using **pandas** and **numpy**.

### Main Steps

1. **Data Loading & Exploration (`task1.py`)** – Load CSVs, check structure, handle missing values, and extract time details.
2. **Data Cleaning & Transformation (`task2.py`)** – Fill missing data, calculate new columns like revenue and profit, and merge datasets.
3. **Data Analysis (`task3.py`)** – Find customer revenue, average order values, and monthly stats.
4. **Basic Exercises** – Simple calculator, Series/DataFrame tasks, and Python fundamentals.

## Libraries

- **numpy** – numerical operations
- **pandas** – data handling and analysis

## Setup

Create and activate a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate   # macOS/Linux
venv\Scripts\activate      # Windows
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python3 task1.py
```
